# Devlicious

Chrome DevTools for Mojolicious:

![DevTools for Mojo](http://cl.ly/image/1C2X3f3v320f/Skjermbilde%202013-01-30%20kl.%2009.32.14.png)

## Synopsis

In one terminal:

```sh
# Download DevTools (only needed once)
$ devlicious download
# Run gateway
$ devlicious run
```

In your application:

```perl
plugin 'Devlicious';
```
